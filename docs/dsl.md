# DSL フォーマット

graph-renderer 独自のグラフ記述言語です。テキストエリアに直接入力して使います。

## 構文

### ノード

```
ID "クラス名 | インスタンス名"
```

- `ID` — 英数字・アンダースコア・ハイフン・コロンが使えます（例: `E22`, `E22_1`, `crm:E22`）
- `"クラス名 | インスタンス名"` — `|` で区切ると上段にクラス名、下段にインスタンス名が表示されます
- `|` を省略すると1段表示になります

### エッジ

```
FROM_ID -> TO_ID : プロパティ名
```

### コメント

ノードにもエッジにも一致しない行は無視されます。`#` など任意の記法でコメントが書けます。

## 例

```
# CIDOC-CRM 記述例
E22 "E22 Human-Made Object | Vase v123"
E42 "E42 Identifier | v123"
E55 "E55 Type | Vase"

E22 -> E42 : P1 is identified by
E22 -> E55 : P2 has type
```

## 備考

- 同じ ID を複数行に書くとノードが重複します（現状チェックなし）
- Google Sheets 連携を使うと、スプレッドシートから自動生成した DSL が挿入されます（[google-sheets.md](google-sheets.md) 参照）
